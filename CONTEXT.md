# LNELABNADEE

Web portal for the Medical Technology Division of Na Dee Hospital (กลุ่มงานเทคนิคการแพทย์โรงพยาบาลนาดี) to manage quality control dashboards, IQC, EQA, and laboratory instrument maintenance checks.

## Language

**LNELABNADEE**:
The name of the Medical Technology Division portal for Na Dee Hospital.
_Avoid_: LABNADEE, viewlabnadee

**IQC**:
Internal Quality Control processes and checks run within the laboratory.
_Avoid_: Internal QC, IQC Check

**EQA**:
External Quality Assessment programs evaluated by external bodies to assess the laboratory's accuracy.
_Avoid_: External QC, EQA Program

**Maintenance**:
Routine scheduled servicing and logging for laboratory analyzer instruments (e.g., AutoLumo, CA-620, Vitross 4600, BC 6200, FUS1000).
_Avoid_: Device Care, Instrument Service

**Language Toggle**:
A header feature allowing users to switch the portal's UI between Thai (TH) and English (EN).
_Avoid_: Lang Switch

**Dashboard**:
The central control page displaying statistics, checklists, navigation, and embed blocks.
_Avoid_: Portal Main

**Instrument Checklist**:
An interactive panel for checking daily/weekly servicing steps of laboratory devices.
_Avoid_: Machine Form

**Metabase Widget**:
A card component displaying charts that can be customized with real Metabase URLs.
_Avoid_: Metabase Iframe

**Theme Toggle**:
A header control allowing users to switch the portal between Light Mode and Dark Mode.
_Avoid_: Dark Toggle

**Responsive Sidebar**:
A navigation drawer that slides out as an overlay on mobile and tablet screens, but remains pinned on desktop.
_Avoid_: Side Menu

**Hamburger Menu**:
A header button visible on small screens to trigger the opening of the Responsive Sidebar.
_Avoid_: Burger Button

**DMSC Auth**:
The Department of Medical Sciences (DMSC) authentication portal tab within the Master List section.
_Avoid_: MOPH Login, MES SSO

**MTNH KPI Monitor**:
The Metabase-powered dashboard panel embedded within the RLU Metabase page, containing sub-menu tabs for Overall KPI, HbA1c, and LDL.
_Avoid_: Metabase iframe, KPI Embed

**Metabase Tutorial**:
The link to the external Google Drive folder containing instructional guides on using Metabase.
_Avoid_: Drive link, PDF guide
