# 1. React Vite SPA Architecture for LNELABNADEE

We decided to build the LNELABNADEE laboratory portal as a Single-Page Application (SPA) using React, Vite, and Vanilla CSS, rather than a multi-page static site. This architecture allows us to organize the 42 sub-pages into a cohesive sidebar navigation system, handle client-side bilingual translation dynamically, and implement high-fidelity mock dashboards with localStorage customization options for embedded Metabase and Google Forms.
